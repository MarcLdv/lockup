# Lockup - Gestionnaire de mots de passe sécurisé

**Lockup** est une application mobile de gestion de mots de passe conçue pour offrir sécurité et confidentialité maximales à l'utilisateur.

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```
